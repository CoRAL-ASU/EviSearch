THREE OPTIONS, THREE USE CASES:
────────────────────────────────

🎯 OPTION A: Extract Value
   → Need one specific value from a PDF
   → Quick extraction (5-10 seconds)
   → Uses API ($0.001)

📋 OPTION B: Extract All Data ✨ NEW
   → Want to see complete 133-column results
   → View baseline experiment extractions
   → Instant, free access to existing data
   → Compare different models/documents

📊 OPTION C: Custom CSV
   → Need custom or non-standard columns
   → Batch extraction from uploaded PDF
   → Uses API ($0.01-0.05)

ALL OPTIONS SHARE:
──────────────────
✓ Same beautiful table/card view
✓ Export to JSON/CSV
✓ Toggle between views
✓ Evidence and location tracking
✓ Professional interface
